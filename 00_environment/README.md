# 00 Environment

## Overview

This document describes the lab environment used for this infrastructure portfolio.

## Host Environment

- Host PC: Panasonic Let's note
- Virtualization: VirtualBox
- Guest OS: CentOS Stream 10

## Purpose

This environment simulates an on-premise Linux server used for healthcare SaaS infrastructure operation and maintenance.

## Main Tools

- Git
- GitHub CLI
- OpenSSH
- systemd
- firewalld
- Apache HTTP Server
- cron
- journalctl

## Notes

This is a personal lab environment.
It does not claim production experience.
